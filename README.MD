# Material ERD
Odoo 14 Simple Materials With Unit Test

## For Running Odoo
.\odoo-bin  --xmlrpc-port=8069 -d name_db --config odoo.conf

## For Unit Testing
.\odoo-bin --test-enable --stop-after-init --config odoo.conf -d name_db -i materials
